real-world business scenarios. The queries demonstrate how structured data
is analyzed to generate insights, support reporting, and drive data-backed
decision-making in operational environments.

The project focuses on transforming raw transactional data into meaningful
metrics using clean, efficient, and scalable SQL logic.
